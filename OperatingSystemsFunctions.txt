An operating system function could include it's memory management. The operating system manages the primary memory. It keeps track of primary memory like which bytes of memory are used by which program. It also keeps track of which memory addresses have already been assigned and which addresses have not been ysed yet. It can also decide the order a procress is granted access to memory and for how long. It will allocate the memory to a certain process when it's requested and then deallocate it after the process ends.
